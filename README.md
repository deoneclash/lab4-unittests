## План тестирования
1. Подготовка тестовых данных:
   - Положительные: (0, 0), (0, 10), (3, 5), (2.5, 4.2), (10**6, 2)
   - Негативные значения: (-1, 5), (5, -1)
   - Неверные типы: "a", None, [1, 2]
2. Ожидаемые результаты:
   - Корректные числовые пары → правильный результат
   - Негативные значения → ValueError
   - Неверные типы → TypeError
3. Выполнение тестов:
   - Запуск через команду:  
     ```
     python -m unittest test_rectangle.py
     ```
4. Сохранение результатов:
   - Результаты фиксируются в консольном выводе (OK / FAIL).

## 2. Описание тестируемого продукта
Модуль `rectangle.py` содержит функцию:
- `area(a, b)` — возвращает площадь прямоугольника.

**Требования:**
- Аргументы должны быть типа `int` или `float`.
- Значения аргументов должны быть неотрицательными.
